## Overview

Derivation 04-peirce-feedback-extension.md (Step 4) established the positivity bound

> |f(lambda_i, lambda_j)| <= sqrt(lambda_i lambda_j)

on the Peirce 1-space mixing coefficient, using a 2x2 Schur complement argument that is manifestly M_2(C)^sa-specific (it represents effects as 2x2 Hermitian matrices). This document proves the same bound holds for **arbitrary finite-dimensional spectral order unit spaces**, using only OUS primitives: compressions, Peirce decompositions, and the positivity of effects.

The proof strategy is approach (b): reduce to 2-level faces.

---

## Proposition (General Positivity Bound)

**Let V be a finite-dimensional spectral OUS with dim >= 2. Let a = sum_i lambda_i p_i be a spectral decomposition of an effect a in [0,1]_V, where {p_i} is a resolution of unity into mutually orthogonal projective units and lambda_i in [0,1]. If the corrected product**

> a & b = sum_i lambda_i C_{p_i}(b) + sum_{i<j} f(lambda_i, lambda_j) P_{ij}(b)

**maps [0,1]_V to [0,1]_V for all effects b, then the mixing function satisfies:**

> **|f(lambda_i, lambda_j)| <= sqrt(lambda_i lambda_j)** for all i < j.

**Equality f = sqrt(lambda_i lambda_j) gives a valid product (the Luders product on M_n(C)^sa).**

## Step 1: 2-Level Faces in a Spectral OUS

**Definition (2-level face).** Given a spectral OUS V and a pair of orthogonal projective units p, q (i.e., p perp q, so C_p(q) = 0) with p + q <= 1, the **2-level face** generated by p and q is:

> F_{p,q} = face(p + q) = range(C_{p+q})

This is the face of the state space determined by the projective unit p + q. By the Alfsen-Shultz compression theory, F_{p,q} is itself a spectral OUS with unit element p + q.

**Key structural fact (Alfsen-Shultz Ch. 7, 9).** In a finite-dimensional spectral OUS, the 2-level face F_{p,q} generated by a pair of minimal orthogonal projective units (atoms) p, q is isomorphic to a spin factor V_n for some n >= 1. For our purposes, the important cases are:

| OUS V | 2-level face | Isomorphism |
|-------|------------|-------------|
| Classical simplex | V_1 (trivial, no V_1 space) | Peirce 1-space = {0} |
| M_n(C)^sa (n >= 2) | V_3 = M_2(C)^sa | dim(V_1(p,q)) = 2 |
| M_n(R)^sa (n >= 2) | V_2 = M_2(R)^sa | dim(V_1(p,q)) = 1 |
| M_n(H)^sa (n >= 2) | V_5 = M_2(H)^sa | dim(V_1(p,q)) = 4 |

The 2-level face F_{p,q} admits the Peirce decomposition:

> F_{p,q} = V_2(p) + V_1(p, q) + V_2(q)

where V_2(p) = span{p}, V_2(q) = span{q} (each 1-dimensional for atoms), and V_1(p, q) is the Peirce 1-space for the pair.

**The crucial property:** V_1(p, q) is nontrivial (i.e., not {0}) whenever V is non-classical and dim >= 2. This is exactly what gives non-commutative OUS their "quantum" character.

## Step 2: Positivity Constraint on a Single 2-Level Face

**Lemma (2-level Schur complement).** Let p, q be orthogonal projective units (atoms) with p + q = e (where e is a projective unit; for a 2-level face, e is its unit). Let a_e = lambda p + mu q be an effect in F_{p,q} with lambda, mu in [0,1]. If the product

> a_e & b = lambda C_p(b) + mu C_q(b) + f(lambda, mu) P_{pq}(b)

maps [0,1]_{F_{p,q}} to [0,1]_{F_{p,q}} for all effects b in [0,1]_{F_{p,q}}, then |f(lambda, mu)| <= sqrt(lambda mu).

**Proof.** The argument is purely order-theoretic and works for ANY spin factor V_n (any n >= 1 with nontrivial V_1).

Any element w of F_{p,q} can be written in the Peirce decomposition as:

> w = alpha p + v + beta q

where alpha, beta in R and v in V_1(p, q). The effect w is in [0, e]_{F_{p,q}} if and only if two conditions hold:

(i) alpha >= 0, beta >= 0 (diagonal entries non-negative), and

(ii) ||v||^2 <= alpha beta (the Peirce 1-space component is bounded by the geometric mean of the diagonal entries).

Condition (ii) is the defining positivity constraint in a spin factor: the state space of V_n is a ball, and the positive cone is the second-order (Lorentz/ice cream) cone.

More precisely, in a spin factor V_n = R + R^n with product (alpha, v) * (beta, w) = (alpha beta + <v,w>, alpha w + beta v), an element (gamma, u) is positive (i.e., >= 0 in the Jordan algebra order) if and only if gamma >= ||u|| (the "ice cream cone" condition). For an element w = alpha p + v + beta q of the 2-level face, writing in the spin factor coordinates with p = (1/2)(e + e_0), q = (1/2)(e - e_0) for a unit vector e_0, we have:

> w = ((alpha + beta)/2) e + ((alpha - beta)/2) e_0 + v

The positivity condition (alpha + beta)/2 >= ||((alpha - beta)/2) e_0 + v|| becomes:

> ((alpha + beta)/2)^2 >= ((alpha - beta)/2)^2 + ||v||^2

which simplifies to:

> alpha beta >= ||v||^2

This confirms condition (ii).

Now apply the product. For b = alpha_b p + v_b + beta_b q with b in [0, e]:

> a_e & b = lambda alpha_b p + f(lambda, mu) v_b + mu beta_b q

This is in [0, e] if and only if:

> (lambda alpha_b)(mu beta_b) >= ||f(lambda, mu) v_b||^2 = f(lambda, mu)^2 ||v_b||^2

i.e.,

> lambda mu alpha_b beta_b >= f^2 ||v_b||^2     ... (*)

This must hold for ALL effects b. Choose b to saturate the positivity bound of the input: take b with ||v_b||^2 = alpha_b beta_b (such b exists -- it is a rank-1 projective unit in V_n). Then (*) becomes:

> lambda mu alpha_b beta_b >= f^2 alpha_b beta_b

For alpha_b beta_b > 0 (which holds for any b with nontrivial Peirce 1-space component), divide:

> lambda mu >= f^2

Therefore:

> **|f(lambda, mu)| <= sqrt(lambda mu)**

This argument used only:
- Peirce decomposition: w = alpha p + v + beta q (OUS primitive)
- Positivity in the OUS order: w >= 0 iff alpha beta >= ||v||^2 (spin factor cone)
- Linearity of compressions and Peirce projections

No matrix representation was invoked. QED. []

## Step 3: From Local Bounds to the Global Bound

**Theorem (General positivity bound).** Let V be a finite-dimensional spectral OUS with dim >= 2. Let a = sum_{i=1}^n lambda_i p_i be a spectral decomposition of an effect, where {p_i} are mutually orthogonal projective units (atoms) forming a resolution of unity. If the corrected product

> a & b = sum_i lambda_i C_{p_i}(b) + sum_{i<j} f(lambda_i, lambda_j) P_{ij}(b)

maps [0,1]_V to [0,1]_V for all b in [0,1]_V, then |f(lambda_i, lambda_j)| <= sqrt(lambda_i lambda_j) for every pair i < j.

**Proof.**

Fix any pair i < j. Consider the 2-level face F_{p_i, p_j} = face(p_i + p_j).

**Step 3a: Restriction to the face.**

The compression C_{p_i + p_j} projects any element of V onto F_{p_i, p_j}. For an effect b in [0,1]_V, the compressed element b' = C_{p_i + p_j}(b) is an effect in [0, p_i + p_j]_{F_{p_i, p_j}} (the effect space of the face).

Conversely, any effect b' in [0, p_i + p_j]_{F_{p_i, p_j}} can be extended to an effect in [0,1]_V (for instance, b' itself, viewed as an element of V, satisfies 0 <= b' <= p_i + p_j <= 1). So every face effect arises as the restriction of some global effect.

**Step 3b: The product restricted to the face.**

Consider a_e = lambda_i p_i + lambda_j p_j, the restriction of a to the 2-level face (i.e., the component of a in F_{p_i, p_j}). For b' in F_{p_i, p_j}:

> a & b' = sum_k lambda_k C_{p_k}(b') + sum_{k<l} f(lambda_k, lambda_l) P_{kl}(b')

Since b' is in F_{p_i, p_j} = range(C_{p_i + p_j}), we have C_{p_k}(b') = 0 for k != i, j (because p_k is orthogonal to both p_i and p_j, hence to p_i + p_j). Similarly, P_{kl}(b') = 0 whenever {k,l} != {i,j}, because the Peirce (k,l) component of an element in face(p_i + p_j) vanishes when k or l is not in {i, j}.

This uses the Peirce orthogonality: the face F_{p_i, p_j} decomposes as V_2(p_i) + V_1(p_i, p_j) + V_2(p_j) under the pair (p_i, p_j), and elements of this face have no components in any V_2(p_k) or V_1(p_k, p_l) for k, l not in {i, j}.

Therefore, for b' in F_{p_i, p_j}:

> a & b' = lambda_i C_{p_i}(b') + lambda_j C_{p_j}(b') + f(lambda_i, lambda_j) P_{ij}(b')

This is exactly the 2-level product from Step 2, with the SAME coefficient f(lambda_i, lambda_j) as the global formula.

**Step 3c: Applying the face constraint.**

Since a & b must be in [0,1]_V for all b in [0,1]_V, and since face effects are a subset of global effects, we have a & b' in [0,1]_V for all b' in [0, p_i + p_j].

Moreover, for b' in F_{p_i, p_j}, the product a & b' (computed above) is itself in F_{p_i, p_j} (because C_{p_i}(b'), C_{p_j}(b'), and P_{ij}(b') are all in the face). The ordering within the face agrees with the global ordering (faces are order-embedded), so a & b' in [0, p_i + p_j]_{F_{p_i, p_j}} is equivalent to a & b' in [0,1]_V restricted to F_{p_i, p_j}.

Therefore the 2-level positivity condition of Step 2 applies, giving:

> **|f(lambda_i, lambda_j)| <= sqrt(lambda_i lambda_j)**

Since the pair (i, j) was arbitrary, this holds for all i < j. QED. []

## Step 4: Sufficiency -- The Bound Is Achievable

**Claim.** If f(lambda_i, lambda_j) = sqrt(lambda_i lambda_j) for all i < j, then the corrected product maps [0,1]_V to [0,1]_V.

**Proof.** On M_n(C)^sa (which is the relevant case after the type exclusion of Phase 05), the corrected product with f = sqrt(lambda_i lambda_j) coincides with the Luders product:

> a & b = sqrt(a) b sqrt(a)

(Proved in 04-peirce-feedback-extension.md Step 10 for M_2(C)^sa; the same matrix calculation extends to M_n(C)^sa because the Luders product on B(H)^sa has the spectral expansion form with f = sqrt(lambda_i lambda_j).)

The Luders product on M_n(C)^sa maps effects to effects (standard result: if 0 <= b <= I and A >= 0, then 0 <= A b A <= ||A||^2 b, so for A = sqrt(a) with 0 <= a <= I we get 0 <= sqrt(a) b sqrt(a) <= b <= I). Therefore f = sqrt(lambda_i lambda_j) gives a valid product.

More generally, for any spectral OUS where the 2-level faces are spin factors: the product with f = sqrt(lambda_i lambda_j) is effect-preserving if and only if on each 2-level face the map b' -> a_e & b' preserves effects in the face. On the face, we computed (Step 2) that the positivity condition is lambda mu alpha_b beta_b >= f^2 ||v_b||^2, and with f^2 = lambda mu this becomes lambda mu alpha_b beta_b >= lambda mu ||v_b||^2, i.e., alpha_b beta_b >= ||v_b||^2, which is exactly the positivity condition on b. So the product preserves positivity on each face, and by the face-decomposition structure of the product (Step 3b), it preserves positivity globally. QED. []

## Step 5: Verification on M_3(C)^sa

To verify the bound beyond M_2(C)^sa, consider M_3(C)^sa with the standard basis of rank-1 projectors {P_0, P_1, P_2} (P_k = |k><k|).

**Setup.** Let a = lambda_0 P_0 + lambda_1 P_1 + lambda_2 P_2 with distinct eigenvalues lambda_0 > lambda_1 > lambda_2 >= 0. The corrected product is:

> a & b = sum_i lambda_i P_i b P_i + f(lambda_0, lambda_1) P_{01}(b) + f(lambda_0, lambda_2) P_{02}(b) + f(lambda_1, lambda_2) P_{12}(b)

where P_{ij}(b) extracts the (i,j) and (j,i) off-diagonal blocks.

**Concrete example.** Take a = diag(3/4, 1/2, 1/4). Consider the 2-level face F_{P_0, P_1} spanned by {P_0, P_1} (the top-left 2x2 block).

A rank-1 projector in this face is:

> b' = (1/2) [[1, 1, 0], [1, 1, 0], [0, 0, 0]]

This has alpha_b = beta_b = 1/2 and ||v_b|| = 1/2 (saturating the bound alpha_b beta_b = ||v_b||^2 = 1/4).

The product restricted to F_{P_0, P_1}:

> a & b' restricted to (0,1) block = [[lambda_0/2, f(lambda_0, lambda_1)/2], [f(lambda_0, lambda_1)/2, lambda_1/2]] = [[(3/4)/2, f/2], [f/2, (1/2)/2]] = [[3/8, f/2], [f/2, 1/4]]

Positivity requires:

> (3/8)(1/4) >= (f/2)^2, i.e., 3/32 >= f^2/4, i.e., f^2 <= 3/8 = lambda_0 lambda_1

So f <= sqrt(3/8) = sqrt(lambda_0 lambda_1). This matches the general bound.

**Check the other faces:**

- F_{P_0, P_2}: f(lambda_0, lambda_2) <= sqrt(3/4 * 1/4) = sqrt(3/16) = sqrt(3)/4.
- F_{P_1, P_2}: f(lambda_1, lambda_2) <= sqrt(1/2 * 1/4) = sqrt(1/8) = 1/(2 sqrt(2)).

With f = sqrt(lambda_i lambda_j) (Luders product), the Luders product on M_3(C)^sa is:

> a & b = sqrt(a) b sqrt(a) = diag(sqrt(3/4), sqrt(1/2), sqrt(1/4)) b diag(sqrt(3/4), sqrt(1/2), sqrt(1/4))

The (0,1) entry of a & b is sqrt(3/4) sqrt(1/2) b_{01} = sqrt(3/8) b_{01}, which saturates the bound f(lambda_0, lambda_1) = sqrt(lambda_0 lambda_1) = sqrt(3/8). Similarly for the other off-diagonal pairs.

2. FACTOR CHECK: For the (0,1) face: 3/32 >= f^2/4. Multiply both sides by 4: 3/8 >= f^2. Check: lambda_0 lambda_1 = (3/4)(1/2) = 3/8. Correct.
3. CONVENTION CHECK: P_k = |k><k| are the standard computational basis projectors. The Peirce (i,j) component P_{ij}(b) extracts the (i,j) and (j,i) entries. Consistent.

## Summary

**Result.** The positivity bound |f(lambda_i, lambda_j)| <= sqrt(lambda_i lambda_j) holds for the corrected sequential product on any finite-dimensional spectral OUS with dim >= 2. The proof requires only:

1. **Peirce decomposition** relative to orthogonal projective units (OUS primitive, Alfsen-Shultz)
2. **Positivity of effects** in the OUS order (alpha beta >= ||v||^2 on spin factor faces)
3. **Face restriction** (effects in a 2-level face are a subset of global effects, and the product restricted to a face uses the same mixing coefficient f)

The argument does not invoke matrix representations, traces, inner products, or any Hilbert space structure. It is purely order-theoretic and works uniformly across all spin factor types (V_n for any n >= 1 with nontrivial Peirce 1-space).

**Logical structure:**

```
Global positivity (a & b in [0,1]_V for all b)
    ==> Face restriction (a & b' in [0,e]_F for all b' in [0,e]_F, each 2-level face F)
        ==> 2-level Schur complement (|f|^2 <= lambda_i lambda_j on each face)
            ==> Global bound (|f(lambda_i, lambda_j)| <= sqrt(lambda_i lambda_j) for all i<j)
```

**Combined with the maximal coherence analysis** (06-maximal-coherence-analysis.md), which shows S5 forces f = sqrt(lambda_i lambda_j), the corrected product is **uniquely determined** on any finite-dimensional spectral OUS: f saturates the positivity bound, and any sub-maximal choice violates associativity for compatible effects.
