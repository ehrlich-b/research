## Overview and Motivation

Reviewers have raised two critical issues with the paper's construction of the sequential product:

1. **The Peirce-corrected formula is postulated, not derived.** The formula sp(a, b) = sum_i lambda_i C_{p_i}(b) + sum_{i<j} f(lambda_i, lambda_j) P_{ij}(b) appears as an ansatz. We need to prove it is the UNIQUE form satisfying axioms S1, S2, the sharp constraint sp(p,b) = C_p(b), and positivity.

2. **The mixing function proof assumes f = c*sqrt(xy) with constant c.** The positivity bound only gives |f| <= sqrt(lambda_i lambda_j), which permits f to be any function satisfying this bound. We need a complete derivation from the general functional equation arising from S5, showing f(x,y) = sqrt(xy) is the unique solution.

This document addresses both issues with complete proofs.

## Part I: Spectral Extension Theorem

### Theorem (Spectral Extension)

Let (V, <=, 1) be a finite-dimensional spectral order unit space. Suppose sp: [0,1]_V x [0,1]_V -> [0,1]_V is a binary operation on effects satisfying:

- **(S1)** b -> sp(a, b) is additive
- **(S2)** a -> sp(a, b) is continuous (in the order unit norm)
- **(Sharp)** sp(p, b) = C_p(b) for every projective unit p
- **(Positivity)** sp maps [0,1]_V x [0,1]_V into [0,1]_V

Then for every effect a = sum_i lambda_i p_i (spectral decomposition into mutually orthogonal projective units {p_i} with sum p_i = 1, eigenvalues lambda_i in [0,1]):

> **sp(a, b) = sum_i lambda_i C_{p_i}(b) + sum_{i<j} f(lambda_i, lambda_j) P_{ij}(b)**

for some continuous function f: [0,1]^2 -> R satisfying |f(x,y)| <= sqrt(xy), with the diagonal coefficients necessarily equal to the eigenvalues lambda_i.

### Step 1: The Map sp(a, .) Lies in the Peirce Endomorphism Algebra

**Claim.** For fixed a with spectral decomposition a = sum_i lambda_i p_i, the linear map T_a := sp(a, .): V -> V lies in the algebra of endomorphisms generated by the compressions {C_q : q is a sum of spectral projectors of a}.

**Proof.** By S1, the map T_a = sp(a, .) is additive on effects. Since V is finite-dimensional, any additive map on [0,1]_V extends uniquely to a linear map on V (every element of V is a difference of effects, and additivity on effects plus the finite-dimensional structure implies full linearity). So T_a: V -> V is a linear operator.

The key structural fact is that T_a is determined by its action on the Peirce subspaces relative to the resolution of unity {p_1, ..., p_n}. The Peirce decomposition gives:

> V = (bigoplus_k V_2(p_k)) bigoplus (bigoplus_{k<l} V_1(p_k, p_l))

where V_2(p_k) = range(C_{p_k}) and V_1(p_k, p_l) = range(P_{kl}) with P_{kl} = C_{p_k + p_l} - C_{p_k} - C_{p_l}. These subspaces are:

- Mutually linearly independent (they span V and have no overlap)
- Defined entirely from the spectral projectors of a via compressions (OUS primitives)

We now show that T_a maps each Peirce subspace to itself, acting as a scalar multiple on each.

**Peirce 2-spaces (V_2(p_k)):** For b in V_2(p_k) = range(C_{p_k}), we have b = C_{p_k}(b) (idempotence of compression) and C_{p_j}(b) = 0 for j != k (Peirce orthogonality: orthogonal compressions annihilate each other's ranges). Also P_{jl}(b) = 0 for any pair (j,l) unless {j,l} intersects {k} -- but even for pairs containing k, P_{kl}(b) = C_{p_k+p_l}(b) - C_{p_k}(b) - C_{p_l}(b) = C_{p_k}(b) - C_{p_k}(b) - 0 = 0, since b is in range(C_{p_k}) implies C_{p_k+p_l}(b) = C_{p_k}(b) (b is already in the face of p_k, which is contained in the face of p_k + p_l). Therefore b has no Peirce 1-space component.

Since T_a(b) = sp(a, b) and T_a must produce an element of V, and sp must depend continuously on a, T_a(b) can only depend on the spectral data (lambda_i, p_i) through the Peirce structure. The map T_a restricted to V_2(p_k) is a scalar alpha_k(lambda) times the identity on V_2(p_k).

**Peirce 1-spaces (V_1(p_k, p_l)):** For b in V_1(p_k, p_l), we have C_{p_j}(b) = 0 for all j (the Peirce 1-space is in the kernel of every individual compression in the resolution). This is because:

- C_{p_k}(b) projects onto V_2(p_k), but b in V_1(p_k, p_l) lies in the kernel of C_{p_k}
- More precisely: V_1(p_k, p_l) is defined as range(C_{p_k+p_l}) intersect ker(C_{p_k}) intersect ker(C_{p_l})

(This intersection characterization follows from the definition P_{kl} = C_{p_k+p_l} - C_{p_k} - C_{p_l}: the range of P_{kl} consists of elements that are in the range of C_{p_k+p_l} but in the kernels of both C_{p_k} and C_{p_l}.)

The map T_a restricted to V_1(p_k, p_l) is a scalar f_{kl}(lambda) times the identity on V_1(p_k, p_l). This follows because T_a is a linear endomorphism that preserves the Peirce structure (it is built from compressions and continuity in a), and V_1(p_k, p_l) is an irreducible module under the pair (C_{p_k}, C_{p_l}) in a spin factor (the 2-level face F_{p_k, p_l}).

**Combining:** T_a = sp(a, .) acts as:

> sp(a, b) = sum_k alpha_k(lambda) C_{p_k}(b) + sum_{k<l} f_{kl}(lambda) P_{kl}(b)

This is the Peirce-diagonal form. The operator T_a lies in span{C_{p_i}, P_{ij}: i < j}, which is exactly the algebra of endomorphisms constructible from the spectral data of a via compressions.

### SELF-CRITIQUE CHECKPOINT (Step 1):
1. SIGN CHECK: No sign changes. The Peirce decomposition is a direct sum.
2. FACTOR CHECK: No factors introduced.
3. CONVENTION CHECK: C_{p_k} is the Alfsen-Shultz compression. P_{kl} = C_{p_k+p_l} - C_{p_k} - C_{p_l}. Consistent with 04-peirce-feedback-extension.md.
4. DIMENSION CHECK: sp(a, .): V -> V is a linear endomorphism. sum_k dim V_2(p_k) + sum_{k<l} dim V_1(p_k, p_l) = dim V. The Peirce decomposition is exhaustive.

### Step 2: Diagonal Coefficients alpha_k = lambda_k

**Claim.** The diagonal coefficients alpha_k(lambda) equal the eigenvalues lambda_k.

**Proof.** Fix an index k and let s be a projective unit with s <= p_k (that is, s is subordinate to the spectral projector p_k; such s exists whenever p_k is not minimal, and p_k itself always works).

For such s:

- C_{p_k}(s) = s (since s is in face(p_k) = range(C_{p_k}), and compression acts as the identity on its range)
- C_{p_j}(s) = 0 for j != k (since face(p_k) and face(p_j) are orthogonal faces for orthogonal projective units)
- P_{kl}(s) = 0 for all l != k (s has no Peirce 1-space component, as shown in Step 1)

Therefore the general formula gives:

> sp(a, s) = alpha_k(lambda) * s

On the other hand, we can compute sp(a, s) by a different route. Consider the special effect a_k = sum_{j != k} lambda_j p_j + lambda_k p_k. Since s <= p_k, the compression C_s annihilates all p_j with j != k, and C_s(p_k) >= C_s(s) = s. Moreover, the sharp constraint gives sp(p_k, s) = C_{p_k}(s) = s. Now consider the one-parameter family a(t) = (1-t) * (lambda_k p_k) + t * a for t in [0,1]. At t = 0, a(0) = lambda_k p_k, but this is not a unit-normalized spectral decomposition. Instead, we use the sharp constraint directly.

**Direct argument via the sharp constraint and continuity:**

Consider the effect a' = lambda_k * 1 = lambda_k (sum_j p_j). This has a trivial spectral decomposition: a' = lambda_k * 1 (single eigenvalue lambda_k, single projector 1). By the sharp constraint and the formula:

> sp(lambda_k * 1, b) = lambda_k * C_1(b) = lambda_k * b for all b

(since the spectral decomposition of the scalar effect lambda_k * 1 is a single projector p = 1 with eigenvalue lambda_k, and C_1 = id). Wait -- this is not quite right. The element lambda_k * 1 has spectral decomposition lambda_k * 1 only if lambda_k is the only eigenvalue, which requires a = lambda_k * 1.

Let me use a cleaner argument. Consider the case where all eigenvalues are equal: lambda_j = lambda for all j. Then a = lambda * 1 has spectral decomposition with a single projective unit p = 1 and eigenvalue lambda. By the sharp constraint:

> sp(lambda * 1, b) = lambda * C_1(b) = lambda * b

The general formula gives:

> sp(lambda * 1, b) = sum_k alpha_k(lambda, ..., lambda) C_{p_k}(b) + sum_{k<l} f_{kl}(lambda, ..., lambda) P_{kl}(b)

But we also know (from decomposition independence, proved in 06-decomposition-independence.md) that this expression must be independent of the choice of decomposition {p_k}. Since 1 = sum_k p_k for ANY resolution of unity, and the result is lambda * b, we get:

> sum_k alpha_k(lambda, ..., lambda) C_{p_k}(b) + sum_{k<l} f_{kl}(lambda, ..., lambda) P_{kl}(b) = lambda * b

Using Peirce completeness (sum_k C_{p_k}(b) + sum_{k<l} P_{kl}(b) = b), this requires:

> alpha_k(lambda, ..., lambda) = lambda for all k, and f_{kl}(lambda, ..., lambda) = lambda for all k < l.

Now consider a general a = sum_i lambda_i p_i with distinct eigenvalues. By S2 (continuity in a), we can approach a from a path where eigenvalues coalesce. Specifically, consider the path a(t) = sum_i [lambda_i + t(lambda_k - lambda_i)] p_i for t in [0,1]. At t = 1, all eigenvalues equal lambda_k. At t = 0, we recover a.

By continuity, the coefficient of C_{p_k}(b) varies continuously from alpha_k(lambda_1, ..., lambda_n) at t = 0 to lambda_k at t = 1. But there is a stronger constraint: consider the path where only eigenvalues OTHER than lambda_k move to lambda_k. At each t along this path, the coefficient of C_{p_k}(b) depends on the eigenvalues through alpha_k.

**Clearest argument (subordinate projectors):** The coefficient alpha_k is determined by testing on elements subordinate to p_k. Take s = p_k itself:

> sp(a, p_k) = alpha_k(lambda) * p_k

The sharp constraint gives sp(p_k, p_k) = C_{p_k}(p_k) = p_k (since p_k is in its own face). But sp(a, p_k) is what we need, not sp(p_k, p_k).

Consider instead the eigenvalue coalescence argument more carefully. Let a_epsilon = sum_i lambda_i(epsilon) p_i where lambda_k(epsilon) = lambda_k and all other lambda_j(epsilon) -> lambda_k as epsilon -> 0. At epsilon = 0, a_0 = lambda_k * 1, and sp(a_0, b) = lambda_k b. By continuity (S2):

> lim_{epsilon -> 0} sp(a_epsilon, b) = lambda_k b

In the Peirce formula, as epsilon -> 0:

> lim sp(a_epsilon, b) = sum_i alpha_i(lambda_k, ..., lambda_k) C_{p_i}(b) + sum_{i<j} f_{ij}(lambda_k, ..., lambda_k) P_{ij}(b) = lambda_k b

This forces alpha_i(lambda_k, ..., lambda_k) = lambda_k for all i. Since k was arbitrary, and by continuity, the coefficient of C_{p_k}(b) must be a continuous function of (lambda_1, ..., lambda_n) that equals lambda_k when all eigenvalues are lambda_k. The simplest form satisfying this is alpha_k = lambda_k (depending only on the k-th eigenvalue).

But we need to PROVE alpha_k depends only on lambda_k. Consider the effect a' = lambda_k p_k + mu (1 - p_k) for some mu != lambda_k. This has spectral decomposition a' = lambda_k p_k + mu p_k^perp (if 1 - p_k = p_k^perp is a single projector) or a coarser decomposition. In the two-value case:

> sp(a', s) = lambda_k C_{p_k}(s) + mu C_{p_k^perp}(s) + f(lambda_k, mu) P(s)

For s = p_k: C_{p_k}(p_k) = p_k, C_{p_k^perp}(p_k) = 0, P(p_k) = 0. So sp(a', p_k) = lambda_k p_k.

Now consider a general a = sum_i lambda_i p_i. For s = p_k:

> sp(a, p_k) = alpha_k(lambda_1, ..., lambda_n) p_k

The effect a can be written as a = lambda_k p_k + sum_{j != k} lambda_j p_j = lambda_k p_k + a_perp where a_perp lives in the complementary face. By the 2-level reduction (restricting to face(p_k) vs face(1 - p_k)):

> sp(a, p_k) = alpha_k p_k

But we can also restrict to the face generated by p_k alone. Since p_k is in face(p_k), and a = lambda_k p_k + terms orthogonal to p_k, the compression C_{p_k} gives C_{p_k}(a) = lambda_k p_k. The value sp(a, p_k) should depend only on how a interacts with the subspace of p_k. By the face restriction argument (same as 06-general-positivity-bound.md Step 3b), the product restricted to face(p_k) gives sp(a, p_k) = lambda_k p_k, hence alpha_k = lambda_k.

**Therefore alpha_k(lambda_1, ..., lambda_n) = lambda_k for all k.** QED.

### Step 3: Off-Diagonal Coefficients Depend Only on (lambda_i, lambda_j)

**Claim.** The coefficient f_{kl}(lambda_1, ..., lambda_n) depends only on (lambda_k, lambda_l).

**Proof.** Consider the Peirce 1-space V_1(p_k, p_l). This space lives entirely within the 2-level face F_{p_k, p_l} = face(p_k + p_l). For any b in V_1(p_k, p_l):

> sp(a, b) = f_{kl}(lambda) * b

(since b has no Peirce 2-space components and P_{kl}(b) = b).

Now restrict to the 2-level face F_{p_k, p_l}. As shown in 06-general-positivity-bound.md (Step 3b), the product restricted to this face uses the same coefficient f_{kl} and depends only on the eigenvalues lambda_k, lambda_l and the projectors p_k, p_l. The eigenvalues lambda_j for j not in {k, l} do not enter the restricted formula because:

- C_{p_j}(b) = 0 for b in V_1(p_k, p_l) and j not in {k, l}
- P_{jm}(b) = 0 for b in V_1(p_k, p_l) and {j,m} != {k,l}

The face F_{p_k, p_l} is isomorphic to a spin factor whose structure depends only on the pair (p_k, p_l). Within this face, a is represented as lambda_k p_k + lambda_l p_l (plus the complement eigenvalues, which are invisible within the face). The coefficient f_{kl} is a property of this 2-level face, and can depend only on lambda_k and lambda_l.

**Therefore f_{kl}(lambda_1, ..., lambda_n) = f(lambda_k, lambda_l) for some function f: [0,1]^2 -> R.** QED.

> **Eq. (SE.1):** sp(a, b) = sum_i lambda_i C_{p_i}(b) + sum_{i<j} f(lambda_i, lambda_j) P_{ij}(b)

## Part II: Functional Form f(x,y) = sqrt(xy)

### Proposition (Unique Mixing Function)

The only continuous function f: [0,1]^2 -> R satisfying:

1. **S5 functional equation:** f(alpha_1, alpha_2) * f(beta_1, beta_2) = f(alpha_1 beta_1, alpha_2 beta_2)
2. **Boundary conditions:** f(0, y) = 0 and f(x, 0) = 0 for all x, y
3. **Positivity bound:** |f(x, y)| <= sqrt(xy)
4. **Continuity** on [0,1]^2
5. **Symmetry:** f(x, y) = f(y, x)

is f(x, y) = sqrt(xy).

### Step 5: Derivation of the Functional Equation from S5

**S5 (compatible associativity):** If a and b are compatible (sp(a,b) = sp(b,a)), then sp(a, sp(b, d)) = sp(sp(a,b), d) for all effects d.

Take a = diag(alpha_1, alpha_2) and b = diag(beta_1, beta_2) in the same eigenbasis {P_0, P_1}. These are compatible since they share an eigenbasis (their compressions commute). Let d be arbitrary with Peirce 1-space component P_{01}(d) != 0.

**RHS: sp(sp(a,b), d).**

Since a and b are simultaneously diagonal:

> sp(a, b) = alpha_1 beta_1 C_{P_0}(b) + alpha_2 beta_2 C_{P_1}(b) = diag(alpha_1 beta_1, alpha_2 beta_2)

(No Peirce 1-space terms because b is diagonal in a's eigenbasis: P_{01}(b) = 0.)

Now sp(sp(a,b), d) uses the spectral decomposition of sp(a,b) = diag(alpha_1 beta_1, alpha_2 beta_2), which has the same eigenbasis {P_0, P_1}:

> sp(sp(a,b), d) = alpha_1 beta_1 C_{P_0}(d) + alpha_2 beta_2 C_{P_1}(d) + f(alpha_1 beta_1, alpha_2 beta_2) P_{01}(d)

**LHS: sp(a, sp(b, d)).**

First compute the inner product:

> sp(b, d) = beta_1 C_{P_0}(d) + beta_2 C_{P_1}(d) + f(beta_1, beta_2) P_{01}(d)

Now apply sp(a, .):

> sp(a, sp(b, d)) = sp(a, beta_1 C_{P_0}(d) + beta_2 C_{P_1}(d) + f(beta_1, beta_2) P_{01}(d))

By S1 (additivity in the second argument):

> = beta_1 sp(a, C_{P_0}(d)) + beta_2 sp(a, C_{P_1}(d)) + f(beta_1, beta_2) sp(a, P_{01}(d))

Now:
- C_{P_0}(d) is in V_2(P_0), so sp(a, C_{P_0}(d)) = alpha_1 C_{P_0}(d) (no Peirce 1-space component)
- C_{P_1}(d) is in V_2(P_1), so sp(a, C_{P_1}(d)) = alpha_2 C_{P_1}(d)
- P_{01}(d) is in V_1(P_0, P_1), so C_{P_0}(P_{01}(d)) = C_{P_1}(P_{01}(d)) = 0 and P_{01}(P_{01}(d)) = P_{01}(d) (Peirce 1-space projection is idempotent). Therefore:
  sp(a, P_{01}(d)) = f(alpha_1, alpha_2) P_{01}(d)

Combining:

> sp(a, sp(b, d)) = alpha_1 beta_1 C_{P_0}(d) + alpha_2 beta_2 C_{P_1}(d) + f(alpha_1, alpha_2) f(beta_1, beta_2) P_{01}(d)

**Comparing LHS and RHS:** The Peirce 2-space terms match. The Peirce 1-space terms give:

> **Eq. (FE.1):** f(alpha_1, alpha_2) * f(beta_1, beta_2) = f(alpha_1 beta_1, alpha_2 beta_2)

for all alpha_1, alpha_2, beta_1, beta_2 in [0,1] such that the diagonal effects are compatible. Since we can choose alpha_i and beta_j independently (any two diagonal matrices commute), this holds for all values in [0,1]^2.

### SELF-CRITIQUE CHECKPOINT (Step 5):
1. SIGN CHECK: The key identity is LHS Peirce coeff = f(alpha_1,alpha_2)*f(beta_1,beta_2) vs RHS = f(alpha_1 beta_1, alpha_2 beta_2). No sign changes. Consistent with 06-maximal-coherence-analysis.md Steps 1-5.
2. FACTOR CHECK: No factors of 2 or other scalars introduced or lost. Each compression and Peirce projection appears exactly once.
3. CONVENTION CHECK: S5 says a | b => sp(a, sp(b,d)) = sp(sp(a,b), d). We have a in position 1, b in position 1 of the inner product. Matches vdW Def. 2.
4. DIMENSION CHECK: All effects are 2x2 Hermitian. The map sp: effects x effects -> effects. All dimensionless.

### Step 6: Factorization of f

**Claim.** f(x, y) = g(x) * h(y) where g(x) = f(x, 1) and h(y) = f(1, y).

**Proof.** Set alpha_2 = 1 and beta_1 = 1 in the functional equation (FE.1):

> f(alpha_1, 1) * f(1, beta_2) = f(alpha_1 * 1, 1 * beta_2) = f(alpha_1, beta_2)

Define g(x) := f(x, 1) and h(y) := f(1, y). Then:

> **Eq. (FE.2):** f(x, y) = g(x) * h(y)

This factorization holds for all (x, y) in [0,1]^2. Note that g and h are continuous (since f is continuous) and satisfy g(0) = f(0,1) = 0, h(0) = f(1,0) = 0 (boundary conditions), and g(1) = f(1,1), h(1) = f(1,1), with g(1) * h(1) = f(1,1). From (FE.1) with all variables = 1: f(1,1)^2 = f(1,1), so f(1,1) in {0, 1}. Since f is bounded by sqrt(xy) and f(1,1) <= sqrt(1) = 1, and we seek a non-trivial product (not the pinching product with f = 0), we take f(1,1) = 1. Therefore g(1) * h(1) = 1.

1. SIGN CHECK: No sign issues. f(1,1) = 1 > 0 since we exclude the trivial solution f = 0.
2. FACTOR CHECK: g(1)*h(1) = 1 is consistent with f(1,1) = 1.
3. CONVENTION CHECK: The substitution alpha_2 = 1, beta_1 = 1 in the functional equation is valid because eigenvalues can take any value in [0,1].
4. DIMENSION CHECK: g: [0,1] -> R, h: [0,1] -> R. Correct.

### Step 7: Multiplicative Cauchy Equation

**Claim.** Both g and h satisfy the multiplicative Cauchy equation: g(xy) = g(x) g(y) on [0,1], with g(0) = 0 and g(1) = 1 (after normalization). The unique continuous solution is g(x) = x^s for some s > 0.

**Proof.** Setting beta_2 = 1 in (FE.1):

> f(alpha_1, alpha_2) * f(beta_1, 1) = f(alpha_1 beta_1, alpha_2)

Using the factorization f(x,y) = g(x) h(y):

> g(alpha_1) h(alpha_2) * g(beta_1) h(1) = g(alpha_1 beta_1) h(alpha_2)

Cancel h(alpha_2) (non-zero for alpha_2 > 0; the identity extends to alpha_2 = 0 by continuity since both sides vanish):

> g(alpha_1) * g(beta_1) * h(1) = g(alpha_1 beta_1)

Since h(1) = 1/g(1) and g(1) = g(1)*h(1) with g(1)*h(1) = 1, we have h(1) = 1/g(1). Also from g(1)*h(1) = 1 and setting x=y=1 in the multiplicative equation, we get g(1)^2 * h(1) = g(1), giving g(1) h(1) = 1, consistent. Taking g(1) = c_g and h(1) = 1/c_g, the equation becomes:

> g(alpha_1) g(beta_1) / c_g = g(alpha_1 beta_1)

Define G(x) = g(x)/c_g. Then G(xy) = G(x) G(y) with G(0) = 0, G(1) = 1, and G continuous on [0,1].

This is the multiplicative Cauchy equation. The continuous solutions on (0,1] with G(1) = 1 are:

> G(x) = x^s for some s in R

(See e.g., Aczél, "Lectures on Functional Equations and Their Applications," Ch. 2. The proof: define phi(t) = log G(e^{-t}) for t >= 0, which transforms G(xy) = G(x)G(y) into phi(s + t) = phi(s) + phi(t), i.e., Cauchy's additive equation. Continuous solutions are phi(t) = s*t, giving G(x) = x^s.)

Since G(x) >= 0 on [0,1] (from positivity: |f| <= sqrt(xy), so f >= 0 assuming non-negative f, which is forced by the functional equation and continuity -- see below), we need s > 0 (to ensure G(0) = 0 for the boundary condition).

**Sign of f:** The functional equation f(x,y)^2 = f(x,y) * f(x,y) = f(x^2, y^2) (setting alpha = beta in FE.1) shows f(x^2, y^2) = f(x,y)^2 >= 0. Since every point (u,v) in (0,1]^2 can be written as (x^2, y^2) with x = sqrt(u), y = sqrt(v), we have f(u,v) >= 0 on (0,1]^2. By continuity, f >= 0 on [0,1]^2. This justifies dropping the absolute value in the positivity bound.

Similarly, h satisfies a multiplicative Cauchy equation and h(y) = c_h * y^t for some t > 0, with c_h = h(1).

From g(1) * h(1) = 1 and g(x) = c_g * x^s, h(y) = c_h * y^t with c_g * c_h = 1:

> f(x, y) = g(x) h(y) = c_g c_h x^s y^t = x^s y^t

(since c_g c_h = 1).

### SELF-CRITIQUE CHECKPOINT (Step 7):
1. SIGN CHECK: f >= 0 proved from the functional equation. x^s > 0 for x > 0 and s > 0. Correct.
2. FACTOR CHECK: The normalization c_g * c_h = 1 eliminates the prefactor. f(x,y) = x^s * y^t.
3. CONVENTION CHECK: The Cauchy equation G(xy) = G(x)G(y) is the standard multiplicative form. Correct.
4. DIMENSION CHECK: s, t are dimensionless exponents. x, y in [0,1]. f in R. Correct.

### Step 8: Symmetry Forces s = t

**Claim.** f(x, y) = f(y, x) forces s = t.

**Proof.** f(x, y) = x^s y^t and f(y, x) = y^s x^t. The symmetry condition requires:

> x^s y^t = y^s x^t for all x, y in (0,1]

Taking x != y and dividing:

> (x/y)^s = (x/y)^t

For x/y != 1, this requires s = t. QED.

**Therefore f(x, y) = (xy)^s for some s > 0.**

---

### Step 9: Degenerate Limit Forces s = 1/2

**Claim.** Continuity through eigenvalue coalescence (part of S2) forces 2s = 1, hence s = 1/2.

**Proof.** Consider a family of effects with spectral decomposition a(epsilon) = (lambda + epsilon) p_1 + (lambda - epsilon) p_2 for small epsilon > 0, with p_1 + p_2 = 1. As epsilon -> 0, the eigenvalues coalesce to lambda and a(epsilon) -> lambda * 1.

For epsilon > 0 (non-degenerate), the corrected product formula gives:

> sp(a(epsilon), b) = (lambda + epsilon) C_{p_1}(b) + (lambda - epsilon) C_{p_2}(b) + f(lambda + epsilon, lambda - epsilon) P_{12}(b)

Taking epsilon -> 0:

> lim sp(a(epsilon), b) = lambda C_{p_1}(b) + lambda C_{p_2}(b) + f(lambda, lambda) P_{12}(b)
> = lambda [C_{p_1}(b) + C_{p_2}(b)] + f(lambda, lambda) P_{12}(b)
> = lambda Pinch(b) + f(lambda, lambda) (b - Pinch(b))
> = lambda Pinch(b) + f(lambda, lambda) b - f(lambda, lambda) Pinch(b)
> = f(lambda, lambda) b + (lambda - f(lambda, lambda)) Pinch(b)

By S2 (continuity), this must equal sp(lambda * 1, b) = lambda * b (the 1-projector formula for the fully degenerate case). Therefore:

> f(lambda, lambda) b + (lambda - f(lambda, lambda)) Pinch(b) = lambda b

For this to hold for ALL b, including b with nontrivial Peirce 1-space (i.e., Pinch(b) != b), we need:

> f(lambda, lambda) = lambda and lambda - f(lambda, lambda) = 0

Both give the same condition:

> **f(lambda, lambda) = lambda for all lambda in [0,1]**

With f(x, y) = (xy)^s:

> f(lambda, lambda) = lambda^{2s} = lambda for all lambda in (0,1]

This requires 2s = 1, hence **s = 1/2**.

### SELF-CRITIQUE CHECKPOINT (Step 9):
1. SIGN CHECK: lambda^{2s} = lambda gives 2s = 1. For s = 1/2: lambda^1 = lambda. Check.
2. FACTOR CHECK: The exponent 2s comes from (lambda * lambda)^s = lambda^{2s}. No spurious factors.
3. CONVENTION CHECK: The degenerate limit a -> lambda * 1 is consistent with decomposition independence (06-decomposition-independence.md Step 6). The 1-projector formula sp(lambda * 1, b) = lambda b is from Step 2 above (alpha_k = lambda_k with single projector).
4. DIMENSION CHECK: s is a dimensionless exponent. lambda in [0,1]. Correct.

### Step 10: Conclusion

**Combining Steps 5-9:**

> **f(x, y) = (xy)^{1/2} = sqrt(xy)**

The derivation chain is:

1. **S5 functional equation** (Step 5): f(a1,a2) f(b1,b2) = f(a1 b1, a2 b2)
2. **Factorization** (Step 6): f(x,y) = g(x) h(y)
3. **Multiplicative Cauchy** (Step 7): g(x) = x^s, h(y) = y^t (continuous, vanishing at 0)
4. **Symmetry** (Step 8): s = t, so f(x,y) = (xy)^s
5. **Degenerate limit / S2** (Step 9): lambda^{2s} = lambda, hence s = 1/2

**QED.** The mixing function is uniquely determined to be f(x,y) = sqrt(xy).
