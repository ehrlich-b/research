## 1. Broken Generators of Spin(9)/Spin(8)

### 1.1 Setup

From Plan 01 (derivations/39-ssb-proof.md):
- H_eff has symmetry G = Spin(9) (dim 36)
- Spontaneous SSB: Spin(9) -> Spin(8) with ordered direction along T_8
- Unbroken subgroup: Spin(8) (dim 28), generated by G_{ab} = [T_a, T_b] for a,b in {0,...,7}
- Broken coset: Spin(9)/Spin(8) = S^8 (dim 8)
- Number of broken generators: n_BG = 36 - 28 = 8

### 1.2 Broken Generator Construction

The 8 broken generators correspond to the coset directions so(9)/so(8). The ordered state selects direction T_8 (by convention). The broken generators are:

$$
Q_a = [T_a, T_8] \quad \text{for } a = 0, 1, \ldots, 7
$$

Using the Clifford anticommutation relation {T_a, T_8} = (1/2)*delta_{a,8}*I_16 = 0 for a != 8:

$$
T_a T_8 = -T_8 T_a \quad (a \neq 8)
$$

Therefore:

$$
Q_a = T_a T_8 - T_8 T_a = 2 T_a T_8
$$

**Properties of Q_a:**
- Q_a is 16x16 real antisymmetric: Q_a^T = (T_aT_8 - T_8T_a)^T = T_8^TT_a^T - T_a^TT_8^T = T_8T_a - T_aT_8 = -Q_a
- Q_a are the generators of the coset so(9)/so(8) in the spinor representation
- Verified numerically: all 8 Q_a have zero antisymmetry error

### 1.3 Commutator of Broken Generators

**Theorem:** [Q_a, Q_b] = -[T_a, T_b] for all a, b in {0,...,7}.

**Proof:**

$$
[Q_a, Q_b] = [2T_aT_8, 2T_bT_8] = 4(T_aT_8T_bT_8 - T_bT_8T_aT_8)
$$

Using $T_8 T_b = -T_b T_8$ (from anticommutation, b != 8):

$$
T_a T_8 T_b T_8 = T_a(-T_b T_8)T_8 = -T_a T_b T_8^2 = -T_a T_b \cdot \tfrac{1}{4}I = -\tfrac{1}{4} T_a T_b
$$

Similarly:

$$
T_b T_8 T_a T_8 = -\tfrac{1}{4} T_b T_a
$$

Therefore:

$$
[Q_a, Q_b] = 4\left(-\tfrac{1}{4}T_a T_b + \tfrac{1}{4}T_b T_a\right) = -(T_a T_b - T_b T_a) = -[T_a, T_b]
$$

**Numerical verification:** Max error ||[Q_a,Q_b] + [T_a,T_b]|| = 0.00e+00 (exact).

**SELF-CRITIQUE CHECKPOINT (step 1):**
1. SIGN CHECK: Factors of (-1): one from T_8T_b = -T_bT_8, one from T_8^2 = (1/4)I. Net effect: 4*(-1/4) = -1 on each term. Subtraction: -(T_aT_b - T_bT_a) = -[T_a,T_b]. Correct.
2. FACTOR CHECK: 4 from (2T_aT_8)(2T_bT_8), 1/4 from T_8^2. Net: 4*(1/4) = 1. Correct.
3. CONVENTION CHECK: {T_a,T_b} = (1/2)*delta*I so T_8^2 = (1/4)*I. Matches lock.
4. DIMENSION CHECK: All quantities are 16x16 matrices. Consistent.

## 2. Watanabe-Murayama Order Parameter Matrix rho_ab

### 2.1 Definition

The Watanabe-Murayama framework (PRL 108, 251602, 2012) classifies Goldstone modes via the matrix:

$$
\rho_{ab} = \langle \text{GS} | [Q_a, Q_b] | \text{GS} \rangle
$$

where |GS> is the symmetry-broken ground state in the thermodynamic limit.

### 2.2 Ground State

The ordered state selects direction T_8. The single-site ground state |psi_0> is an eigenvector of T_8 with eigenvalue +1/2 (maximum eigenvalue, consistent with ferromagnetic ordering).

From the Clifford algebra T_8^2 = (1/4)*I_16, T_8 has eigenvalues +/- 1/2, each with multiplicity 8. The +1/2 eigenspace E_+ has dimension 8.

In the thermodynamic limit, the ferromagnetic ground state is a product state |psi_0>^(x)N. The relevant rho_ab is:

$$
\rho_{ab} = \langle \psi_0 | [Q_a, Q_b] | \psi_0 \rangle
$$

for any specific choice of |psi_0> in E_+ (the thermodynamic limit breaks the residual Spin(8) symmetry within E_+).

### 2.3 Vanishing of rho_ab (Analytical Proof)

**Theorem:** rho_ab = 0 for all a, b in {0,...,7}, for ANY choice of ground state |psi_0> in E_+.

**Proof:**

From Section 1.3: [Q_a, Q_b] = -[T_a, T_b].

The matrix [T_a, T_b] = T_aT_b - T_bT_a is **real antisymmetric**:
- T_a, T_b are real symmetric (Clifford generators in Cl(9,0) with positive definite signature)
- (T_aT_b - T_bT_a)^T = T_b^TT_a^T - T_a^TT_b^T = T_bT_a - T_aT_b = -[T_a, T_b]

The ground state |psi_0> is a **real vector** (eigenvector of the real symmetric matrix T_8, so can always be chosen real).

For any real antisymmetric matrix A and real vector v:

$$
v^T A v = (v^T A v)^T = v^T A^T v = -v^T A v \implies v^T A v = 0
$$

Therefore:

$$
\rho_{ab} = -\langle \psi_0 | [T_a, T_b] | \psi_0 \rangle = -\psi_0^T [T_a, T_b] \psi_0 = 0
$$

This holds for ANY real vector, not just eigenvectors of T_8. The vanishing is EXACT, not approximate.

**SELF-CRITIQUE CHECKPOINT (step 2):**
1. SIGN CHECK: rho = -<[T,T]> = -(0) = 0. No sign ambiguity.
2. FACTOR CHECK: No numerical factors -- vanishing is structural.
3. CONVENTION CHECK: Real symmetric T_a from Cl(9,0). Matches convention lock.
4. DIMENSION CHECK: rho_ab is 8x8 dimensionless matrix. Correct.

### 2.4 Numerical Verification

Computed rho_ab numerically for:
1. A specific eigenvector of T_8 with eigenvalue +1/2: max|rho_ab| = 0.00e+00
2. All 8 eigenvectors in E_+: max|rho_ab| = 0.00e+00
3. 20 random superpositions within E_+: max|rho_ab| = 2.78e-17 (machine zero)

The rho_ab matrix is:

$$
\rho_{ab} = \begin{pmatrix} 0 & 0 & \cdots & 0 \\ 0 & 0 & \cdots & 0 \\ \vdots & & \ddots & \vdots \\ 0 & 0 & \cdots & 0 \end{pmatrix}_{8 \times 8}
$$

rank(rho) = 0.

### 2.5 Physical Mechanism for Vanishing

The vanishing of rho_ab has a clear algebraic origin: the on-site Hilbert space R^16 is a REAL representation of Spin(9). The generators T_a are real symmetric matrices, and their commutators [T_a, T_b] are real antisymmetric. For any real state, the expectation value of a real antisymmetric operator vanishes identically.

**Comparison with the standard Heisenberg ferromagnet (SU(2), spin S):**
- The SU(2) generators S_x, S_y, S_z are COMPLEX Hermitian matrices
- [S_x, S_y] = iS_z is Hermitian (not antisymmetric)
- <up|[S_x, S_y]|up> = i<S_z> = i*S != 0
- Result: Type-B magnons with omega ~ k^2

The difference is fundamental: SU(2) spin representations are COMPLEX, so the commutator [S_x, S_y] = iS_z is Hermitian and can have nonzero expectation values. Our Spin(9) spinor representation is REAL, so the commutator [T_a, T_b] is antisymmetric and has vanishing expectation values on any real state.

**Key insight:** Despite the ground state being ferromagnetic (aligned spins), the Goldstone modes are Type-A (not Type-B) because the representation is real. The "ferromagnetic" label refers to the spin alignment pattern, but the Goldstone dynamics depends on the representation type:
- Complex representation: rho != 0, Type-B (quadratic dispersion)
- Real representation: rho = 0, Type-A (linear dispersion)

## 3. Watanabe-Murayama Counting

### 3.1 The Counting Rule

The Watanabe-Murayama counting rule (PRL 108, 251602, 2012; Ann. Rev. Cond. Mat. Phys. 11, 2020) states:

$$
n_A + 2 n_B = n_{BG}
$$

where:
- $n_{BG}$ = number of broken generators = 8 (from Spin(9)/Spin(8))
- $n_B$ = (1/2) rank($\rho_{ab}$) = number of Type-B Goldstone modes
- $n_A$ = number of Type-A Goldstone modes

### 3.2 Application to Spin(9)/Spin(8)

From Section 2: rank($\rho_{ab}$) = 0.

$$
n_B = \tfrac{1}{2} \times 0 = 0
$$

$$
n_A = n_{BG} - 2 n_B = 8 - 0 = 8
$$

**Verification of sum rule:** $n_A + 2 n_B = 8 + 0 = 8 = n_{BG}$. Exact.

### 3.3 Goldstone Mode Classification

$$
\boxed{n_A = 8, \quad n_B = 0}
$$

All 8 Goldstone modes are **Type-A** (linear dispersion).

## 4. Dispersion Relations

### 4.1 Type-A Dispersion

For the 8 Type-A Goldstone modes, the low-energy dispersion relation is:

$$
\omega_a(\mathbf{k}) = c_s |\mathbf{k}| + O(k^2), \quad a = 1, \ldots, 8
$$

where $c_s$ is the spin-wave velocity.

**Dimensional check:** $[\omega] = [\text{energy}]$, $[k] = [1/\text{length}]$, $[c_s] = [\text{length} \times \text{energy}] = [\text{velocity}]$ (in natural units with hbar = 1). Consistent.

### 4.2 Spin-Wave Velocity

For the O(N) sigma model on Z^d with N = 9:

$$
c_s = \sqrt{\frac{2 J \rho_s}{\chi_\perp}}
$$

where $\rho_s$ is the spin stiffness and $\chi_\perp$ is the transverse susceptibility. Both depend on temperature and can be computed from quantum Monte Carlo or spin-wave theory.

At leading spin-wave order for the O(9) model: $c_s = J a \sqrt{2/N} = J a \sqrt{2/9}$ where $a$ is the lattice spacing (= 1 in our units). With the QMC-calibrated value from Phase 34 (for O(3)): $c_s \approx 1.659 J a$. The O(9) value will differ but has the same scaling.

### 4.3 Absence of Type-B Modes

There are NO Type-B modes. The dispersion does NOT contain any quadratic branches $\omega \sim k^2$. The entire low-energy spectrum is relativistic.

## 5. Impact on Lorentz Emergence

### 5.1 The Critical Question

The central question of Plan 02 was: do the Goldstone modes of the Spin(9) -> Spin(8) breaking have linear (Type-A) or quadratic (Type-B) dispersion?

**Answer:** ALL 8 modes are Type-A with linear dispersion $\omega = c_s |k|$.

### 5.2 Consequences for Lorentz Invariance

The sigma model on S^8 = Spin(9)/Spin(8) has action:

$$
S_\sigma = \frac{\rho_s}{2} \int d^d x \, \partial_\mu \mathbf{n} \cdot \partial_\mu \mathbf{n}, \quad \mathbf{n} \in S^8
$$

with relativistic dispersion $\omega = c_s |k|$. After Wick rotation and rescaling $x_0 = c_s \tau$, this becomes the O(9)-invariant Euclidean action:

$$
S_E = \frac{\rho_s}{2 c_s} \int d^{d+1} x \, \partial_\mu \mathbf{n} \cdot \partial_\mu \mathbf{n}
$$

which has full O(d+1) rotational invariance. In the physical $d = 3$ case, O(4) Euclidean rotation symmetry $\to$ SO(3,1) Lorentz symmetry upon continuation back to Minkowski signature.

**Verdict:** Lorentz emergence is CONSISTENT with the Goldstone mode spectrum. The speed $c_s$ becomes the emergent speed of light.

### 5.3 Comparison with What Could Have Gone Wrong

If rho_ab had been nonzero (as expected for a "generic" ferromagnet with complex representation):
- rank(rho) = 8 would give n_B = 4, n_A = 0: all modes Type-B, Lorentz fails completely
- rank(rho) = 4 would give n_B = 2, n_A = 4: mixed spectrum, partial Lorentz
- rank(rho) = 2 would give n_B = 1, n_A = 6: mostly Type-A but not fully relativistic

The fact that rank(rho) = 0 is the BEST CASE for Lorentz emergence. It is not fine-tuned -- it is a structural consequence of the representation being real.

## 6. Benchmark: O(3) Heisenberg Ferromagnet

### 6.1 SU(2) Spin-1/2 (Complex Representation)

The standard Heisenberg ferromagnet has SU(2) symmetry with generators S_x, S_y, S_z that are COMPLEX Hermitian matrices ($S_x = (1/2)\sigma_x$, etc.). The SSB pattern is SU(2) -> U(1), with broken generators S_x, S_y.

$$
\rho_{12} = \langle \uparrow | [S_x, S_y] | \uparrow \rangle = \langle \uparrow | i S_z | \uparrow \rangle = i/2 \neq 0
$$

rank(rho) = 2 (full rank for the 2x2 case), so n_B = 1, n_A = 0. The single Goldstone mode is Type-B with $\omega \sim k^2$. This is the standard ferromagnetic magnon.

### 6.2 O(3) Vector Model (Real Representation)

The O(3) nonlinear sigma model has generators L_x, L_y, L_z as 3x3 real antisymmetric matrices acting on R^3. The SSB pattern is O(3) -> O(2), ordered along z.

For a REAL vector $\mathbf{e}_3 = (0,0,1)^T$ and the REAL ANTISYMMETRIC generator $L_z$:

$$
\rho_{12} = \mathbf{e}_3^T [L_x, L_y] \mathbf{e}_3 = \mathbf{e}_3^T L_z \mathbf{e}_3 = 0
$$

(since $L_z$ is antisymmetric and $\mathbf{e}_3$ is real). rank(rho) = 0, n_A = 2, n_B = 0. Both Goldstone modes are Type-A with $\omega = c_s |k|$.
